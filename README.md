# Neovim configuration
This is a configuration folder for neovim located in `$HOME/.config/`. It uses packer as its package manager.

## Structure 
```
nvim
├── init.lua
├── LICENSE
├── lua
│   └── usr
│       ├── colorscheme.lua
│       ├── keymaps.lua
│       ├── options.lua
│       └── plugins.lua
├── plugin
│   └── packer_compiled.lua
└── README.md
```
